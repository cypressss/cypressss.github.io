3.winter sleeps

<?php
error_reporting(0);
require __DIR__.'/lib.php';

if(isset($_GET['time'])){

    if(!is_numeric($_GET['time'])){
        echo 'The time must be number.';

    }else if($_GET['time'] < 60 * 60 * 24 * 30 * 2){
        echo 'This time is too short.';

    }else if($_GET['time'] > 60 * 60 * 24 * 30 * 3){
        echo 'This time is too long.';

    }else{
        sleep((int)$_GET['time']);
        echo $flag;
    }

    echo '<hr>';
}

highlight_file(__FILE__);

题目要求输入时间5184000-7776000之间，这得等死都出不来。可以用科学记数法绕过5.184e6-7.776e6之间都可以过。

5.URL filtering

上源码

<?php
error_reporting(0);
require __DIR__."/lib.php";

$url = urldecode($_SERVER['REQUEST_URI']);
$url_query = parse_url($url, PHP_URL_QUERY);

$params = explode("&", $url_query);
foreach($params as $param){

    $idx_equal = strpos($param, "=");
    if($idx_equal === false){
        $key = $param;
        $value = "";
    }else{
        $key = substr($param, 0, $idx_equal);
        $value = substr($param, $idx_equal + 1);
    }

    if(strpos($key, "do_you_want_flag") !== false || strpos($value, "yes") !== false){
        die("no hack");
    }
}

if(isset($_GET['do_you_want_flag']) && $_GET['do_you_want_flag'] == "yes"){
    die($flag);
}

highlight_file(__FILE__);

注意到里面有个函数parse_url(),用来解析URL返回其组成部分

源码要求的是返回？之后的数据，
尝试输入http://urlfiltering.solveme.peng.kr/?do_you_want_flag=yes

果不其然返回no hack。

但如果是http://urlfiltering.solveme.peng.kr///?do_you_want_flag=yes
就可以返回flag

因为parse_url()在解析URL时有漏洞，通过 ///x.php?key=value 就可以返回flase

7.array2string

<?php
error_reporting(0);
require __DIR__.'/lib.php';

$value = $_GET['value'];

$username = $_GET['username'];
$password = $_GET['password'];

for ($i = 0; $i < count($value); ++$i) {
    if ($_GET['username']) unset($username);
    if ($value[$i] > 32 && $value[$i] < 127) unset($value);
    else $username .= chr($value[$i]);

    if ($username == '15th_HackingCamp' && md5($password) == md5(file_get_contents('./secret.passwd'))) {
        echo 'Hello '.$username.'!', '<br>', PHP_EOL;
        echo $flag, '<hr>';
    }
}

highlight_file(__FILE__);

这里要求value的值在ascii码的可见范围之外，而且username是由value拼接成‘15th_HackingCamp’
Chr()自动mod256，所以只要输上述字符的ASCII+256就可以通过。

str="15th_HackingCamp"
for i insstr:
    a=ord(i)
print a+256

密码也不是随便输的，源码提示访问./secret.passwd可以得到密码

11.hell js

F12看到一堆js混淆的代码，直接F12放控制台里跑不出来。
jsfuck有个翻译规则是

eval => []["filter"]["constructor"]( CODE )()

所以说题目源码应该是放在CODE部分里面，所以只要把CODE那部分抠下来放在控制台里跑就可以
复制jsfuck到sublime里方便找CODE

跑出个hello，应该后面还有，继续找

flag就写在源代码里

12.name check

截取部分关键代码

 $res = $sql->query("
        SELECT 
        MAX('0','1','{$name}') LIKE 'a%', 
        INSTR('{$name}','d')>0, 
        MIN('{$name}','b','c') LIKE '__m__', 
        SUBSTR('{$name}',-2)='in'
    ;");

if(
        $row[0] + $row[1] + $row[2] + $row[3] !== 4 ||
        array_sum($row) !== 4 
    ){
        echo 'Auth failed';
        goto quit;
    }

    echo $flag;

1) 输入的name要以a开头

2) name的第二个字母要是d

3) m要放在中间

4) 最后两个字母是in

连起来不就是admin吗，但是前面做了限制

四个参数加起来要等于4，
就是说要满足上面的四个条件

sqlite3里连接字符串使用的是 || 而不是 + 所以可以用 || 绕过

payload：

http://namecheck.solveme.peng.kr/?name=adm'||'in
